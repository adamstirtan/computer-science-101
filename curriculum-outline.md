# Computer Science Curriculum Outline

## 1. Programming Foundations

- Variables, loops, conditionals
- Functions
- Basic data types
- Simple projects

## 2. Computational Thinking

- Problem decomposition
- Algorithm design basics

## 3. Math Foundations

- Logic
- Sets
- Functions
- Graph basics

## 4. Data Structures

- Arrays, lists
- Stacks, queues
- Trees, graphs

## 5. Algorithms

- Searching & sorting
- Recursion
- Problem-solving patterns

## 6. Complexity Analysis

- Big-O notation
- Trade-offs

## 7. Regular Expressions + Automata

- Regex basics
- DFA/NFA concepts
- Pattern matching theory

## 8. Evolutionary Algorithms

- Genetic algorithms and population-based search
- Fitness functions, selection, crossover, mutation
- Exploration vs. exploitation strategies
- Practical optimization scenarios

## 9. Neural Networks

- Perceptrons and feedforward networks
- Backpropagation and gradient descent
- Training, validation, and overfitting prevention
- Applications in classification, regression, and representation learning

### Further topics

- APIs
- Reading binary files
- Introduction to frameworks
- History of computation
- How a CPU works
- Parts of a computer
- What is the full-stack?
